# The Ponderosa

A small team building practical software and AI workflows.

This repo hosts a simple public teaser site used for applications and lightweight outreach.

## What we do (high level)

- Build and ship user-facing products
- Measure outcomes (instrumentation, experiments, iteration)
- Care about polish and usability

We keep this site intentionally minimal and non-proprietary.

## No secrets

This repo is designed to be public.

- No API keys or credentials are stored here.
- No `.env` files are required to build.

## Development

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
npm run start
```
